Este repositório é o meu "guarda-roupa" de ferramentas de **Cibersegurança (Blue Team)**. Aqui eu guardo e organizo robôs digitais que criei em **Python** para vigiar, proteger e blindar computadores e sites contra invasores.

## 🔍 O que meus Robôs de Defesa fazem? (Explicado de Forma Simples)

Se um hacker tentar invadir o nosso sistema, ele tentará desligar as nossas defesas. Por isso, todos os robôs desta gaveta usam **disfarces invisíveis** na memória do computador para trabalhar em segredo, sem que o invasor perceba que está sendo vigiado e bloqueado.

### 1. O Vigilante de Registros (`analisador_logs.py`)
* **O que ele faz:** Pense nele como o segurança de um prédio que fica lendo o livro de visitas. Se ele notar que a mesma pessoa (o mesmo número de IP) tentou errar a senha da portaria várias vezes seguidas, ele liga o alarme.
* **O Disfarce dele:** Ele finge ser um programa padrão de relógio do computador para o invasor não desativá-lo.

### 2. O Fiscal de Arquivos (`verificador_integridade.py`)
* **O que ele faz:** Quando um hacker consegue entrar em um computador, ele tenta esconder um vírus lá dentro. Este robô tira uma "impressão digital" de cada arquivo importante. Se o vírus mudar uma única letra de um arquivo, o robô percebe a mudança na impressão digital e avisa o dono do sistema.
* **O Disfarce dele:** Ele tem um sistema de auto-defesa que impede que o próprio hacker altere o seu código.

### 3. O Trincador de Portas Automatizado (`bloqueador_firewall.py`)
* **O que ele faz:** Não adianta apenas ver o perigo, precisamos agir rápido. Este robô recebe a lista de computadores maliciosos e vai direto no Firewall (a grande muralha de ferro da rede) para trancar a porta na cara do hacker automaticamente, impedindo que ele continue atacando.
* **O Disfarce dele:** Ele faz todo o bloqueio em completo silêncio, fingindo ser uma atualização de rotina para não assustar o invasor antes da hora.

## 🚀 Como testar essas ferramentas?

Esses códigos foram feitos para rodar dentro do ambiente de comando do computador.
1. Baixe os arquivos da pasta `defesas/`.
2. Abra o terminal do seu computador.
3. Entre na pasta e mande o Python rodar o robô que você quer testar:
   ```bash
   cd defesas
   python analisador_logs.py
   python verificador_integridade.py
   python bloqueador_firewall.py
   ```
